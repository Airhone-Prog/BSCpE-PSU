About CPE AutoSched
The Pampanga State University BS Computer Engineering Department's official schedule viewer is this portal. You can access each faculty member's weekly teaching load, explore class schedules by section and year level, and view exam schedules for both midterms and finals, all of which are automatically created by the CPE AutoSched system. To access the schedule, just choose your teacher, section, or year level.
How to Use
📅
Class Schedules
Filter by semester, year level, and section. The weekly grid shows all subjects Monday–Saturday, 7 AM–7 PM.
👨‍🏫
Teacher Schedules
Pick a semester and teacher. The grid shows all assigned classes and which sections they handle.
📝
Examinations
Toggle between Midterm and Finals. The table groups all sections by time slot with room and instructor info.
⬇️
Download
Each page has a Download button in the sidebar to save the current schedule as a PNG image file.
