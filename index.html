About This Portal
AutoSched — PSU CpE Department · AY 2025–2026
What is AutoSched?

AutoSched is the automated scheduling system of the BS Computer Engineering Department at Pampanga State University. It generates conflict-free class and exam schedules for all sections, assigns teachers and rooms, and publishes the result here — no login needed.

This portal reads pre-exported JSON data files from the scheduling engine. The data files are loaded from the data/ folder at the root of this repository.

How to Use
📅
Class Schedules
Filter by semester, year level, and section. The weekly grid shows all subjects Monday–Saturday, 7 AM–7 PM.
👨
Teacher Schedules
Pick a semester and teacher. The grid shows all assigned classes and which sections they handle.
📝
Examinations
Toggle between Midterm and Finals. The table groups all sections by time slot with room and instructor info.
⬇️
Download
Each page has a Download button to save the current schedule view as a PNG image file.
